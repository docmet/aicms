Connect your WordPress site to MyStorey to control content via Claude, ChatGPT, or any MCP-compatible AI.

== Description ==

MyStorey lets your AI assistant write, update, and publish WordPress posts and pages through conversation — no clicking through menus required.

**How it works:**
1. Install this plugin
2. Create an Application Password in your WordPress profile
3. Register your site at mystorey.io
4. Paste the MCP URL into Claude.ai or ChatGPT

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mystorey-connector/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings → MyStorey AI to follow the setup steps
